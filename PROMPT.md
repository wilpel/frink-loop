# Task Prompt

Write your task description here. This file is read by Frink Loop if no command-line prompt is provided.

## Example

```markdown
## Task
Add a user authentication system to the application.

## Requirements
- Use JWT tokens for session management
- Implement login, logout, and register endpoints
- Add middleware for protected routes
- Store user passwords with bcrypt hashing
- Include basic rate limiting on auth endpoints

## Acceptance Criteria
- All endpoints return appropriate HTTP status codes
- Tests pass for all auth flows
- No TypeScript errors
```
